
# User Account & Data Deletion Process

**Echos of Mind**

## 1. Deletion Request Method

To request permanent deletion of their account and all associated data, users must:

* Send an email from their **registered (verified) email address**
* Subject line: `Account Deletion Request`
* Send to: `support@echosofmind.com` (replace with actual email)

We only accept deletion requests from the verified email linked to the account.

This prevents unauthorized deletion.

---

## 2. Verification Process

When a request is received:

1. Confirm the sender email matches an existing account.
2. If it matches → acknowledge request within 48 hours.
3. If it does not match → respond asking the user to email from their registered address.

No identity documents required.
Email match is sufficient.

---

## 3. 7-Day Cooling Period

After acknowledgment:

* A **7-day waiting period** begins.
* During this period:

  * Account remains accessible.
  * User may cancel deletion by replying to the same thread.

Why 7 days?

* Prevents accidental deletion.
* Prevents malicious account takeover deletion.
* Provides user reconsideration window.

No new data is processed for insight generation during this window if technically feasible.

---

## 4. Final Deletion (Day 7)

On Day 7:

We permanently delete:

* User profile data
* Journal entries
* Mood check-ins
* Voice data (if stored)
* Behavioral signal history
* Authentication tokens
* Linked metadata

Deletion applies to:

* Primary database
* Backups (within standard rotation cycle)
* Analytics tied to identifiable user ID

Backups are purged within standard infrastructure retention cycle (e.g., 30 days max).

After deletion:

* Account cannot be recovered.
* Data cannot be restored.
* Baseline history is permanently erased.

User receives confirmation email.

---

## 5. What May Be Retained (Minimal Security Logs)

We may retain minimal logs strictly for:

* Legal compliance
* Fraud prevention
* Abuse detection

These logs:

* Do not contain journal content
* Do not contain emotional analysis
* Are not used for profiling
